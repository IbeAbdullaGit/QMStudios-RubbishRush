PER-OBJECT MOTION BLUR DEMO
--------------------------------------------------------------------------------
This demo accompanies the tutorial which can be found at

	http://john-chapman-graphics.blogspot.co.uk/2013/01/per-object-motion-blur.html

	
The provided executables were compiled on Windows 7 with GCC 4.7.1 (MinGW) and 
were tested on the following platforms:
	
	Windows 7		NVIDIA GeForce GTX 570 (driver 310.90)
	Windows XP		NVIDIA GeForce GTX 260 (driver 310.90)

	
Demo not running? Bug reports/feedback are appreciated via
	
	contact@john-chapman.net
	
	
Only demo-specific source code is provided. If you wish to compile the code, you
need common.zip which can be found at
	
	http://john-chapman-graphics.blogspot.co.uk/p/demos.html
	
as well as SFML 2.0, available from

	http://www.sfml-dev.org/download.php

CONTROLS
--------------------------------------------------------------------------------
F1							Toggle overlay
F2							Toggle helpers
F3							Toggle render texture visualisation
F4							Toggle vsync

F8							Reload textures
F9							Reload/compile/link shaders

W,A,S,D					Move forward, left, backward, right
RMOUSE					Mouse look
LCTRL + RMOUSE			Zoom

C							Cycle current camera, clip camera
LSHIFT + C				Cycle current clip camera only

B							Cycle environment map
